这个项目用于存放正式文章、持续更新的个人笔记和项目记录。

## 技术栈

- Astro：生成静态网站。
- Markdown/MDX：写文章和笔记。
- GitHub Pages：托管生成后的静态页面。
- GitHub Actions：在 push 后自动构建和部署。

## 当前阶段

先完成最小可用版本，保证本地可预览、GitHub 可部署、内容结构清晰。
