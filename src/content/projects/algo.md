这个项目是我维护的算法模板库，用于集中保存常用数据结构、图论、字符串、数论和多项式相关模板。

项目仓库：[jendolkk/algo](https://github.com/jendolkk/algo)

## 当前内容

仓库中主要包含 C++ 模板，也有少量 Python 和 Rust 辅助代码。现有文件覆盖了这些方向：

- 图论：最短路、强连通分量、最大流、最小费用最大流、匹配、2-SAT、拓扑排序。
- 数据结构：并查集、树状数组、线段树、重链剖分。
- 字符串：KMP、Aho-Corasick、Z 函数、后缀数组。
- 数学：扩展欧几里得、欧拉函数、组合数、Lucas、矩阵、高斯消元。
- 多项式与数论变换：FFT、NTT、多项式模板。
- 常用工程片段：快速 IO、随机数、编译 pragma、测试数据生成脚本。

## 展示方式

这个项目页不需要把所有模板源码复制到网站里。更合适的做法是：

- 网站里写项目介绍、目录结构和使用场景。
- 具体源码仍然放在 GitHub 仓库中维护。
- 后续可以挑选稳定模板，单独写成文章或笔记，解释适用条件、复杂度和易错点。
